Quick Start 
===========

Assuming you have Python already, install the package:

.. tabs::

   .. tab:: pip

      .. prompt:: bash

         pip install censys


      If you do not have pip installed, get it `here <https://pip.pypa.io/en/stable/installation/>`_.

   .. tab:: on Kali Linux

      .. prompt:: bash

         sudo apt install python3-censys

Configure your credentials:

.. tabs::

   .. tab:: Search API

      .. prompt:: bash

         censys config

   .. tab:: ASM API

      .. prompt:: bash
      
         censys asm config
